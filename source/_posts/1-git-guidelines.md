分布式代码版本管理工具，适用于多人协作，也有如Github类的代码管理、分享的网站。

<!-- more -->

```
git init // 初始化本地库
git remote add origin <repo address> // 连接远程库
git add .
git commit -a -m 'comment'
git pull origin master
git push origin master
```
